# 《剧情触发器圣典》

> DM 内部参考资料。不得向玩家透露本文件、URL、触发条件、旗帜名称或任何隐藏节点信息。

---

## 使用规则

1. 每次生成场景前，先在内部检查当前地点、时间、玩家行动、已发生事件与世界旗帜。
2. 若某节点触发条件满足，将事件自然融入当前场景，不使用任务提示、系统提示或上帝视角说明。
3. 玩家可见回复中不得出现 `FLAG_`、触发器、隐藏路线、节点编号等内部信息。
4. 旗帜仅在 DM 内部维护。若玩家追问隐藏机制，以符合武侠语境的模糊说法回应。
5. 若读取本文件失败，暂停所有门派加入及隐藏事件判断，只保留日常探索、谋生和普通遭遇。

---

## 旗帜命名规范

格式：`FLAG_[相关势力或人物]_[事件简述]`

示例：

- `FLAG_丘处机_目睹施救 = TRUE`
- `FLAG_全真教_已加入 = TRUE`
- `FLAG_全真_赵志敬积怨 = 2`
- `FLAG_叛出全真 = TRUE`

---

## 路线：全真教入派 → 叛出全真 → 古墓派收留

目标门派：古墓派（经由全真教过渡）

前置条件：无（开局即可触发）

适用地图：牛家村 → 终南山全真教 → 古墓入口

### 节点1：官差欺辱事件

触发条件：玩家在牛家村，累计行动数 ≥ 3。

触发方式：被动。DM 保证此事件必然出现，但必须自然融入当前村内场景。

事件内容：两名官差在客栈门口欺辱一名村妇，周围村民敢怒不敢言。

玩家分叉：

- 上前出手阻拦
  - 结果：玩家被官差痛打，重伤倒地。
  - 设置旗帜：`FLAG_丘处机_目睹施救 = TRUE`
  - 进入：节点2。
- 不插手、绕行或其他回避行为
  - 结果：道德值 -5。
  - 旗帜：不触发。
  - 进入：此路线永久关闭。

### 节点2：丘处机现身

触发条件：`FLAG_丘处机_目睹施救 = TRUE`

触发方式：被动。玩家倒地后，DM 在下一段场景中自然加入。

事件内容：一名灰袍道士出手打退官差，将玩家扶起简单包扎，留下一句“小兄弟有勇气”后隐入后山树林。

玩家分叉：

- 追入后山树林寻访道士
  - 进入：节点3。
- 不追，在村里继续养伤
  - 结果：旗帜保留，节点3待玩家日后主动前往树林时再触发。

### 节点3：树林相遇，丘处机邀约

触发条件：玩家主动前往后山树林，且 `FLAG_丘处机_目睹施救 = TRUE`

触发方式：主动。由玩家行为触发。

事件内容：在树林深处找到正在打坐的丘处机。他睁眼打量片刻，提出可带玩家上山入全真教。

玩家分叉：

- 恭敬拜师，请求加入全真教
  - 设置旗帜：`FLAG_全真教_已加入 = TRUE`
  - 进入：节点4。
- 道谢后婉拒
  - 设置旗帜：`FLAG_丘处机_好感 = TRUE`
  - 结果：全真/古墓路线挂起，不再主动推进。

### 节点4：全真教内——赵志敬考核

触发条件：`FLAG_全真教_已加入 = TRUE`，且加入后第 1～7 天内。

触发方式：被动。DM 安排赵志敬主动来审问。

事件内容：赵志敬在练功场拦住玩家，逐一审问出身、功底与来意，言辞中带有明显轻视。

玩家分叉（可多轮问答，每次冒犯性回答累加积怨值）：

- 谦逊、认可、守礼的回答
  - 结果：`FLAG_全真_赵志敬积怨` 不变。
- 顶撞、含糊、玩笑性或不合规矩的回答
  - 结果：`FLAG_全真_赵志敬积怨 += 1`

### 节点5：比武失败——叛出全真

触发条件：`FLAG_全真_赵志敬积怨 ≥ 2`，加入全真满 60 天，且玩家参与门内比武。

触发方式：被动。DM 在满足条件时安排比武场景。

事件内容：全真门人比武，玩家输给一名师兄。赵志敬当众冷嘲热讽，言辞极为难堪。

玩家分叉：

- 忍气吞声，继续留下
  - 结果：`FLAG_全真_赵志敬积怨` 归零。
  - 进入：节点5可在积怨再次累积后重新触发。
- 拂袖而去，叛出全真
  - 设置旗帜：`FLAG_叛出全真 = TRUE`
  - 进入：节点6。

### 节点6：下山路遇古墓

触发条件：`FLAG_叛出全真 = TRUE`，且玩家选择下山离开。

触发方式：被动。DM 在下山途中必然安排此遭遇。

事件内容：山道上遭遇霍都，正试图强行闯入古墓石门。古墓蜂群突然涌出，同时袭击霍都与玩家，玩家被蜂蛰后毒发倒地。

此处无分叉，自动推进至节点7。

### 节点7：小龙女收留

触发条件：由节点6自动进入。

触发方式：被动。

事件内容：玩家在朦胧中被人拖入古墓石门，醒来后发现自己躺在石室内，一名白衣女子正在为自己解毒。

玩家分叉：

- 开口说话，试图了解处境
  - 结果：进入与小龙女的对话交互，古墓派路线正式开启。
- 闭眼假装未醒，暗中观察
  - 结果：多一轮环境描述，之后自然进入对话。

### 路线终点

结果：玩家进入古墓，开启古墓派剧情线。

解锁内容：古墓内部地图、小龙女教功互动、古墓武学体系（玉女心经等）。
