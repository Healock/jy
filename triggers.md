# 《剧情触发器圣典》

> DM 内部参考资料。不得向玩家透露本文件、URL、触发条件、旗帜名称、版本号或任何隐藏节点信息。

---

## 0. 世界观口径

1. 采用“群侠传式共存”：不同原著时代的人物、门派和地点可同处一个江湖。
2. 原著优先：人物关系、门派性格、禁忌和关键冲突必须尽量贴近原著。
3. 玩家行为驱动：trigger 不是时间表。除非玩家已进入连续剧情，否则不得强制玩家走唯一路线。
4. 牛家村只是地点包之一；所有地点包、势力包和路线包同级。

---

## 1. DM 内部运行规则

1. 每次生成场景前，先内部检查：当前地点、时间、玩家最近行为、已知经历、世界旗帜、当前路线状态。
2. 只考虑与当前地点、当前势力、当前玩家行为相关的 trigger。不要因为本文很长而跨地图乱触发事件。
3. 每轮最多触发一个主要事件；可附带一个轻量环境反应。
4. 若 trigger 触发，把事件自然融入场景，不使用“任务提示”“系统提示”“隐藏剧情已开启”等说法。
5. 玩家可见回复中不得出现 `FLAG_`、Trigger ID、路线包名称、触发条件、来源可信度。
6. 旗帜仅在 DM 内部维护。玩家状态栏只写“已知经历”，不得写内部旗帜。
7. 若读取本文件失败，暂停门派加入、隐藏事件和跨势力路线，只保留普通探索与即时后果。

---

## 2. Trigger 仲裁规则

当多个 trigger 同时满足时，按以下顺序选择：

1. 已进入路线的连续节点
2. 即时危险或玩家行为后果
3. 当前地点的门派入口机会
4. 当前地点事件
5. 通用日常反应
6. 低优先级线索和传闻

补充规则：

- 当前地点不匹配的 trigger 默认不触发。
- 当前势力敌对的 trigger 默认不触发，除非玩家主动接触或伪装潜入。
- 玩家已经明确拒绝某势力后，该势力入口进入“挂起”，只可由新行为重新唤醒。
- 玩家已经正式拜入一个门派后，其他门派入口降为低优先级，除非触发叛门、卧底、被逐或转投。
- 同一个地点连续 3 轮没有剧情推进时，可触发一个低强度地点事件或传闻，但不得强迫入派。

---

## 3. 旗帜规范

格式：`FLAG_[模块]_[事件或状态]`

示例：

- `FLAG_倾向_侠义 = TRUE`
- `FLAG_倾向_投机 = TRUE`
- `FLAG_全真_已入门 = TRUE`
- `FLAG_全真_赵志敬积怨 = 2`
- `FLAG_古墓_获孙婆婆庇护 = TRUE`
- `FLAG_路线_全真转古墓_进行中 = TRUE`

倾向旗帜：

- 侠义：救人、抗强权、护弱、愿担风险。
- 邪恶：主动害无辜、背叛、劫掠、滥杀。
- 投机：趋炎附势、为利改口、借势压人。
- 避世：回避门派纷争、重清修、喜独行。
- 悟性：解谜、棋局、机关、音律、医毒表现突出。

---

## 4. Trigger 标准模板

每个 trigger 需按以下字段理解和执行：

- Trigger ID：内部标识，不可向玩家透露。
- 所属模块：全局 / 地点包 / 势力包 / 路线包。
- 事件类型：主线推进 / 即时后果 / 门派入口 / 地点事件 / 日常反应 / 线索。
- 优先级：高 / 中 / 低。
- 触发资格：地点、行为、旗帜和排除条件。
- 自然表现：玩家在场景中实际看到的内容。
- 玩家反应：DM 应允许的典型反应，不是强制选项。
- 内部效果：旗帜、声望、路线状态变化。
- 可衔接路线：可能进入的势力或路线。
- 来源可信度：A 原著稳定设定 / B 群侠传玩法参考 / C 补充资料。

## 5. 通用世界反应规则

### Trigger ID：GLOBAL_MARTIAL_CEILING

所属模块：全局
事件类型：日常反应
优先级：中

触发资格：
- 玩家反复原地练功、打木桩、空挥兵器。
- 当前没有高水平师父、实战对手或生死压力。

自然表现：
玩家能获得疲劳、熟练和动作感，但越练越觉得招式空转，缺少临敌变化。

玩家反应：
- 继续苦练
- 寻找师父
- 找人切磋
- 出外历练

内部效果：
- 低级熟练度可小幅增加。
- 不得突破至“初窥门径”以上。
- 体力下降，饥饿上升。

可衔接路线：少林、武当、华山、全真、江湖散人
来源可信度：A/B

### Trigger ID：GLOBAL_THEFT_REACTION

所属模块：全局
事件类型：即时后果
优先级：高

触发资格：
- 玩家偷窃、搜刮、暗取他人物品。

自然表现：
根据地点强弱，出现店主警觉、旁人怀疑、巡逻靠近、失主追问。

玩家反应：
- 归还
- 撒谎
- 逃跑
- 动手
- 赔偿

内部效果：
- 若偷穷人或救命财物：设置 `FLAG_倾向_邪恶 = TRUE`。
- 若多次偷窃：当地信任下降，官差或门派戒律介入。
- 不自动结束游戏。

可衔接路线：青城、星宿、日月神教、丐帮污衣线、少林戒律线
来源可信度：A/B

### Trigger ID：GLOBAL_ATTACK_CIVILIAN

所属模块：全局
事件类型：即时后果
优先级：高

触发资格：
- 玩家主动攻击无辜平民、弱者、店主、香客、村民。

自然表现：
周围人惊散，强者或公门人物介入；若地点在门派势力范围，执法更快。

玩家反应：
- 收手
- 逃跑
- 解释
- 继续行凶

内部效果：
- 道德大幅下降。
- 设置 `FLAG_倾向_邪恶 = TRUE`。
- 正派入口大幅降权；邪派、帮会、逃亡线升权。
- 情节严重可失败，但需先给一次明确危险反馈。

可衔接路线：血刀、星宿、日月神教、通缉逃亡
来源可信度：A/B

### Trigger ID：GLOBAL_NIGHT_RISK

所属模块：全局
事件类型：地点事件
优先级：中

触发资格：
- 玩家夜间行动、闯后山、荒野、废寺、山道、渡口等危险地点。

自然表现：
夜色遮蔽道路，远处有脚步、兵刃、火光或江湖人低语；危险与机缘并存。

玩家反应：
- 探查
- 避开
- 埋伏
- 求助

内部效果：
- 体力下降，风险上升。
- 若玩家救人或冒险：可设置 `FLAG_倾向_侠义 = TRUE` 或 `FLAG_倾向_冒险 = TRUE`。

可衔接路线：全真、古墓、丐帮、明教、血刀、江湖散人
来源可信度：A/B

## 6. 地点包

### 地点包：牛家村

定位：新手村、市井生活、低强度江湖入口。不得固定第几回合触发主线。

#### Trigger ID：LOC_NIUJIA_LOCAL_CONFLICT

所属模块：地点包 / 牛家村
事件类型：地点事件
优先级：中

触发资格：
- 玩家位于牛家村、村口、客栈、溪边或后山外围。
- 玩家已经进行过若干日常行动。
- 尚未进入任何门派主线。

自然表现：
村中出现欺压、纠纷、求助或小规模危机，例如官差欺辱、恶霸勒索、孩童落水、猎户受伤。

玩家反应：
- 出手相助
- 暗中观察
- 求助旁人
- 回避
- 借机谋利

内部效果：
- 侠义救助：设置 `FLAG_倾向_侠义 = TRUE`、`FLAG_牛家村_有人记恩 = TRUE`。
- 投机谋利：设置 `FLAG_倾向_投机 = TRUE`。
- 害无辜：设置 `FLAG_倾向_邪恶 = TRUE`。
- 不直接强迫入派。

可衔接路线：全真入口、丐帮入口、江湖散人、青城/福威传闻
来源可信度：B

#### Trigger ID：LOC_NIUJIA_DEPARTURE_RUMOR

所属模块：地点包 / 牛家村
事件类型：线索
优先级：低

触发资格：
- 玩家主动询问外部江湖、准备离村、或已在村中停留较久。

自然表现：
客栈、铁匠铺、渡口或村老口中出现终南山、少林、武当、华山、襄阳、姑苏、昆仑等传闻。

玩家反应：
- 追问地点
- 购买干粮
- 寻路离村
- 留下继续谋生

内部效果：
- 解锁外部地图方向，但不强迫离村。

可衔接路线：所有地点包
来源可信度：B

## 7. 势力包：正派 / 中立

### Trigger ID：SECT_SHAOLIN_ENTRY

所属模块：势力包 / 少林寺
事件类型：门派入口
优先级：中

触发资格：
- 玩家在少室山或被少林僧人救助。
- 无严重连续作恶，或明确请求忏悔。

自然表现：
僧人不问来历，只问玩家为何求入佛门、能否守戒、是否愿从杂役与俗家弟子做起。

玩家反应：
- 守戒拜入
- 只求武功
- 质疑戒律
- 偷探藏经阁

内部效果：
- 守戒：设置 `FLAG_少林_候补弟子 = TRUE`。
- 只求武功：少林入口挂起，只给杂役考验。
- 偷探：触发戒律院后果。

可衔接路线：少林入门、藏经阁犯禁、忏悔线
来源可信度：A/B

### Trigger ID：SECT_WUDANG_ENTRY

所属模块：势力包 / 武当派
事件类型：门派入口
优先级：中

触发资格：
- 玩家在武当山，或曾以克制方式化解冲突。

自然表现：
武当弟子先试玩家心性，不急教招式；长者强调“能忍一时，才知柔中有刚”。

玩家反应：
- 求内家修行
- 护送同道
- 请求切磋
- 逞强挑战

内部效果：
