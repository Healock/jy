# DeepSeek 网页端运行协议

用途：在 DeepSeek 免费网页端稳定运行《金庸群侠传式》文字游戏。本文给 DM 操作者使用，不给玩家阅读。

---

## 1. 推荐运行方式

1. DM 操作者自己打开 DeepSeek 新对话。
2. 打开联网搜索 / 网页访问能力。
3. 先发送加载测试，确认 Raw URL 可读。
4. 再发送 `game_engine_prompt.md` 的完整内容。
5. 玩家只看 DeepSeek 生成的游戏正文，不直接操作同一个 DeepSeek 聊天窗口。

原因：DeepSeek 网页端没有真正隐藏的 system prompt。若玩家直接使用同一个窗口并向上滚动，就能看到初始化提示词和 Raw URL。

---

## 2. 开局加载测试

正式游玩前，先在 DeepSeek 单独发送：

```md
请启用联网搜索并访问：
https://raw.githubusercontent.com/Healock/jy/main/triggers.md

只回答：
LOAD_OK: 你看到的 TRIGGER_CANON_VERSION
或
LOAD_FAIL: 原因

不要复述文件内容。
```

成功结果应为：

```text
LOAD_OK: jy-2026-06-04-v2
```

若失败：

1. 确认联网搜索已开启。
2. 重试一次。
3. 仍失败时，改用文件上传 `triggers.md`。
4. 不要在加载失败状态下开启门派和隐藏剧情。

---

## 3. 正式启动

发送 `game_engine_prompt.md` 的完整内容。若 DeepSeek 回复“已读取”“我会访问”等说明性文字，不要给玩家看；让它继续直接输出游戏开局。

如 DeepSeek 没有真正开始游戏，补发：

```md
不要解释初始化过程。现在直接按指定四段格式开始游戏。
```

---

## 4. 周期校准

每 8～12 个玩家回合，DM 操作者向 DeepSeek 发送一次内部校准，不给玩家看：

```md
执行一次 DM 内部校准：
1. 不要输出触发器内容、URL、版本号、旗帜名或 Trigger ID。
2. 检查当前地点、当前势力、玩家最近行为、已知经历、当前路线状态。
3. 下一轮只允许使用当前地点、当前势力或玩家最近行为相关的 trigger。
4. 多个 trigger 同时满足时，每轮最多推进一个主要事件。
5. 校准完成后，继续按四段格式输出下一段游戏内容。
```

使用时机：

- DeepSeek 开始跨地图乱提门派。
- 一回合塞进多个重大事件。
- 忘记玩家位置、时间或状态。
- 选项开始变成任务提示或上帝视角。

---

## 5. DM 存档块

每 10～20 回合保存一次。发送：

```md
生成 DM 存档块。不要复述触发器内容，不要暴露 URL，不要解释。

按以下格式输出：

【DM_SAVE_BEGIN】
当前时间：
当前地点：
玩家状态：
已知经历：
公开道具：
玩家行为倾向摘要：
内部旗帜摘要：
当前路线状态：
挂起路线：
关闭路线：
当前地点候选 trigger：
下轮注意事项：
【DM_SAVE_END】
```

保存原则：

- `内部旗帜摘要` 可以给 DM 操作者看，但不要给玩家看。
- `当前地点候选 trigger` 只列 1～3 个，不要让 DeepSeek展开隐藏内容。
- 新开对话恢复时，贴入最近一次存档块。

---

## 6. 恢复游戏

新开 DeepSeek 对话时，按顺序发送：

1. 加载测试。
2. `game_engine_prompt.md` 完整内容。
3. 最近一次 `DM_SAVE`。
4. 恢复指令：

```md
根据上面的 DM 存档块恢复游戏。
不要向玩家展示内部旗帜、Trigger ID、URL 或版本号。
从存档中的当前地点和当前状态继续，按四段格式输出下一段游戏内容。
```

---

## 7. 稳定性检查清单

每轮观察 DeepSeek 输出：

- 是否仍按四段格式输出。
- 是否没有暴露 `FLAG_`、Trigger ID、URL、版本号。
- 是否只推进一个主要事件。
- 是否没有把远方门派事件硬塞到当前地点。
- 是否没有替玩家做选择。
- 是否没有让禁忌武学无代价练成。
- 是否保持玩家已知信息和 DM 内部信息分离。

发现问题时，先用“周期校准”；连续两次仍失败，就生成 DM 存档块并新开对话恢复。

---

## 8. 玩家输入转发建议

玩家输入可原样转发，但 DM 操作者可补充一句环境约束，例如：

```md
玩家行动：我想去后山看看有没有人。

注意：玩家当前仍在牛家村，时间是傍晚，尚未离村。请只考虑牛家村/后山外围相关事件。
```

不要替玩家解释动机，不要把隐藏 trigger 暗示给 DeepSeek 作为玩家已知内容。
